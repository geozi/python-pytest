# python-pytest
![Unit testing demo project]( https://img.shields.io/badge/Demo_project-blue)

### About the project
The project showcases the use of PyTest on Python code.

### Prerequisites
To run the tests, the following will be needed: 
* Python 3.10.12 (or higher)
* PyTest 8.3.3
